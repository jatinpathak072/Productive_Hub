# 🚀 Productive Hub

A modern and responsive productivity dashboard built with **HTML, CSS, and JavaScript**. Productive Hub helps users stay organized with task management, daily planning, goal tracking, weather updates, motivational quotes, and a Pomodoro timer—all in one place.

## 🌟 Features

- 🕒 Real-time Digital Clock & Date
- 👋 Dynamic Greeting (Morning, Afternoon, Evening, Night)
- 🌤️ Live Weather using OpenWeatherMap API
- 💬 Random Motivational Quotes
- ✅ Todo List
  - Add, Edit & Delete Tasks
  - Mark Tasks as Completed
  - Task Counter
  - Data saved using Local Storage
- 📅 Daily Planner
  - Editable Schedule
  - Task Completion Tracking
  - Progress Saved in Local Storage
- 🎯 Daily Goals
  - Add Goals
  - Mark Goals as Completed
  - Delete Goals
- 📊 Circular Progress Tracker
  - Shows Completed, Pending & Total Tasks
- 🍅 Pomodoro Timer
  - Start
  - Stop
  - Reset
- 🌙 Light/Dark Mode
- 💾 Local Storage Support

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
- DummyJSON Quotes API
- Local Storage
